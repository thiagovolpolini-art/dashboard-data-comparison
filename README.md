# 📊 DataVision AI

Dashboard web moderno para importar, visualizar e comparar dados de arquivos CSV e Excel.

O projeto permite transformar planilhas em gráficos, indicadores e análises visuais, além de oferecer um assistente de Inteligência Artificial para responder perguntas sobre os dados importados.

## 🌐 Demonstração online

Acesse o projeto publicado:

[🔗 Abrir DataVision AI](https://thiagovolpolini-art.github.io/dashboard-data-comparison/)

## 📌 Sobre o projeto

O DataVision AI foi desenvolvido para facilitar a análise de dados por pessoas que não possuem experiência com programação ou ferramentas avançadas de Business Intelligence.

A aplicação possui uma interface moderna e responsiva, permitindo visualizar informações, comparar tabelas e identificar padrões nos dados de maneira mais simples.

O projeto pode ser utilizado em diferentes áreas, como:

- Vendas;
- Finanças;
- Estoque;
- Marketing;
- Futebol;
- Gestão de produtos;
- Comparação de resultados.

## ✨ Principais funcionalidades

- Importação de arquivos CSV e Excel;
- Visualização dos dados em tabelas;
- Identificação das colunas disponíveis;
- Criação de gráficos interativos;
- Comparação entre diferentes arquivos;
- Comparação entre categorias e períodos;
- Indicadores com informações importantes;
- Filtros para análise dos dados;
- Interface responsiva;
- Modo escuro;
- Assistente com Inteligência Artificial;
- Publicação automática pelo GitHub Actions.

## 🤖 Assistente de Inteligência Artificial

O projeto possui uma integração experimental com a API da Anthropic.

O usuário pode fazer perguntas em linguagem natural sobre os dados carregados, por exemplo:

```text
Qual produto teve o maior número de vendas?

Qual foi o mês com maior faturamento?

Compare os resultados de janeiro e fevereiro.

Quais são os cinco jogadores com mais gols?

Qual time possui a maior média de gols?

Qual categoria apresentou maior crescimento?
